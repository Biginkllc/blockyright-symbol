## Description

This is the foundational Blockyright symbol, using a solid black "b" enclosed in a black circle on a transparent background.  

It is designed for use across:

- documentation and readme files  
- digital publications  
- parables and written works  
- domain banners (light backgrounds)  
- project provenance files  
- GitHub repository headers  
- IPFS archival uploads  
- provenance proof documents  

---

## Recommended Usage

Use this symbol when:

- the background is white or light  
- you need maximum clarity and contrast  
- a “classic copyright-style” look is preferred  
- using GitHub, Markdown, or PDF documents  
- adding a clear BR signature to visual media
