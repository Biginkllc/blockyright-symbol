# Stamp-Style Symbols

This folder will contain stamp-style blockyright marks, including:

- Rough-edged ink stamp looks
- Vintage seal styles
- Monochrome stamp impressions

These are especially useful for:

- Vintage photo provenance
- Bookplates and ex libris designs
- Legal-style or archival documents
- Artistic or historic collections

Files will be added here as they are created.
